# QA_System

The system consists of two components: the document retriever module for Ô¨Ånding the most relevant article in Wikipedia and the machine comprehension model (Bidirectional Attention Flow) for extracting the answer from the retrieved document.

### Requirements
The following libraries need to be installed:
- allennlp (https://github.com/allenai/allennlp/tree/v0.4.0)
- pytorch (http://pytorch.org/)
- spacy's models (https://spacy.io/usage/models)
- wikipedia (https://pypi.python.org/pypi/wikipedia)
- rake-nltk (https://pypi.python.org/pypi/rake-nltk)
- scikit-learn (http://scikit-learn.org/stable/install.html)

### How to run the program
```
./run_qa
```
